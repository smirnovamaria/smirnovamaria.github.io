Модель хищник-жертва

Модель. Описание модели

Простейшая модель взаимодействия двух типов “хищник - жертва” - модель Лотки-Волтерры. Эта двухвидовая модель основана на следующих предположениях:

  1. Численность популяции добычи x и хищников y зависит только от времени

  2. При отсутствии взаимодействия количество видов изменяется по модели Мальтуса, с увеличением количества добычи и падением числа хищников.

  1. Естественная смертность добычи и естественная рождаемость хищника считаются незначительными

  2. Эффект насыщения обеих популяций не рассматривается

  3. Темпы увеличения численности добычи снижаются пропорционально изобилию хищников

В этой модели x - количество добычи, y - количество хищников.

Алгоритм

Наша модель хищник-жертва имеет только два начальных состояния:

  1. Стационарное состояние (рис. 1, точка А)
  2. Состояние, имеющее отклонение от стационарного (рис. 1, точка В)

При небольшом изменении модели происходит следующее:

$$ \frac{dx}{dt} = -ax(t) + bx(t)y(t) $$

$$\frac{dy}{dt} = cy(t) - dx(t)y(t)$$

Здесь х - количество добычи, у - число хищников. а - отношение естественной смертности хищников, b - отношение естественного прироста добычи, с - отношение прироста хищников, d - отношение смертности добычи.

Для решения этой задачи мы использовали систему для математических вычислений GNU Octave. GNU Octave - свободная система для математических вычислений, использующая высокоуровневый MATLAB - совместимый язык.

set(figure,'Name','Lotka-Volterra models of Predator-Prey Relationships','NumberTitle','off')

function dx=f(x,t)
	dx(1) = -0.2*x(1) + 0.05*x(1)*x(2);
	dx(2) = 0.5*x(2) - 0.02*x(1)*x(2);
endfunction


x0=[5;10]; 
t = [0: 0.1: 400];
y = lsode("f", x0, t);
y1 = y(:, 1);
y2 = y(:, 2);

sPoint = [ 0.5/0.02; 0.2/0.05 ]

subplot(2,2, 1:2);
  	hold on
  	plot(sPoint(1),sPoint(2), 'o')
	plot(y1, y2, "linewidth", 1 , 'm');
  	legend("Stationary point", "LotkaVolterra equations")
	xlabel("Predators");
	ylabel("Preys");
	title("Predator-Prey model");
subplot (2, 2, 3);
	plot(t, y1, "linewidth", 1 , 'g');
	xlabel("Time");
	ylabel("Number of predators");
	title("Number of changes predator populations");
subplot (2, 2, 4);
	plot(t, y2, "linewidth", 1 , 'c');
	xlabel("Time");
	ylabel("Number of preys");
	title("Number of changes prey populations");
